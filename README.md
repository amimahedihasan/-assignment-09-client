# 🩺 DocAppoint

A modern and responsive Doctor Appointment Booking web application built with **Next.js App Router**.  
DocAppoint helps users easily search doctors, view detailed profiles, and book appointments with a smooth and user-friendly experience.

---

## 🌐 Live Demo

🔗 https://assignment-09-chi.vercel.app/

---

## 🚀 Key Features

### 👨‍⚕️ Doctor Management
- Display all available doctors
- View detailed doctor information
- Search doctors using query parameters
- Dynamic doctor profile pages

### 🔐 Authentication System
- Secure authentication-based access control
- Protected private routes
- User login & registration system

### 📅 Appointment Features
- Book doctor appointments
- Manage appointment-related data
- Responsive appointment UI

### ⚡ Performance & UI
- Fast server-side rendering with Next.js
- Fully responsive for mobile, tablet, and desktop
- Clean modern UI design
- Reusable component architecture

### 🔗 API Integration
- Fetch doctor data from APIs
- Dynamic data handling
- Optimized API requests
