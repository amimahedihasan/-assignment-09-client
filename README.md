# 🩺 DocAppoint

DocAppoint is a modern Doctor Appointment Booking web application built with **Next.js**.  
It allows users to search doctors, view details, and manage appointment-related features with a fast and responsive UI.

---

## 🚀 Features

- 🔍 Doctor search using query params
- 👨‍⚕️ Display all available doctors
- 🔐 Authentication-based access control
- ⚡ Fast server-side rendering (Next.js App Router)
- 📦 API integration for fetching data
- 📱 Fully responsive design
- 🧠 Server & Client Component architecture

---

## 🛠️ Tech Stack

- Next.js (App Router)
- React
- Tailwind CSS
- Node.js (API layer)
- React Icons
